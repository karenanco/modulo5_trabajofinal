## Base de datos relacional para un e-commerce que gestiona el inventario, perfiles de usuario (con roles), categorías y el flujo de pedidos con transacciones seguras.

# Orden de Ejecución
1. schema.sql: Define la estructura.
2. seed.sql: Carga datos iniciales.
3. queries.sql: Ejecuta consultas de prueba.
4. transaction.sql: Prueba el flujo de venta.
